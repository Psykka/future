# 📝 Notas Rápidas — Índice

> [!info] Como funciona
> Espaço para ideias, hipóteses e pensamentos que surgiram durante o projeto mas ainda não justificam uma pesquisa formal.
> Cada nota tem um ciclo de vida explícito — nada fica em "semente" para sempre.

---

## 🔄 Ciclo de vida das notas

| ciclo | significado |
|-------|-------------|
| `semente` | ideia registrada, não avaliada ainda |
| `avaliando` | em análise ativa (pesquisa, council, pre-mortem) |
| `planejando` | decidido ir em frente, definindo como |
| `ativa` | em implementação |
| `arquivada` | boa ideia, não é o momento |
| `descartada` | avaliada e rejeitada, com motivo registrado |
