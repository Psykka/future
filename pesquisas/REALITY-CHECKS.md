# 🔍 Reality Checks — Índice

> [!info] Como funciona
> Cada nota atua como sócio estratégico pragmático: identifica o que é hipótese vs. evidência,
> qual o gargalo real (tecnologia, distribuição, confiança, regulação ou capital) e o que NÃO
> deve ser feito ainda. Os arquivos ficam em `pesquisas/reality-checks/`. Use `/reality-check` para criar uma nova análise.
